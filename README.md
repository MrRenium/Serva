# Serva - PHP & Static Web Server

Serva is a simple, fast, and colorful local web server written in Python.  
It supports HTML, CSS, JS, images, and PHP files. Automatically serves `index.php` or `index.html` if present.

## Features

- Colorful detailed request logs
- HTML, CSS, JS, and images support
- PHP support via PHP CLI
- Automatic `index.php` / `index.html` serving
- Request stats every minute
